Genesis of the Blogging App

This blogging app was originally a Codeup project - a free-standing, multi-user blogging application, written and implemented in SpringBoot and Java.

I've repurposed it a bit - both to host my vanity site but also to allow me to blog on this site.

This is, of course, both better and worse than what it originally was. It's quite a bit more optimized this way, quite a bit faster, and quite a bit more stylish than it originally started out.

But it's also quite a bit less robust than it originally was - again, you can no longer create an account as a new user and begin blogging - I've locked up the functionality that would allow anyone else to use this app.

All of which to say - if you see any vestigal bits laying around, that's why. And if you wondered why I didn't build more functionality on to it - that's also why.
